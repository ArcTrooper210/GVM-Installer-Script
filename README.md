# GVM Installer Script
Bash installer for [Greenbone Community Edition](https://greenbone.github.io/docs/latest/22.4/source-build/index.html).

## Supported OS
* Ubuntu 22.04 LTS

## Usage
Run with `sudo bash gvmInstaller.sh`
